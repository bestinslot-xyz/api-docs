Get Sats Activity

Returns ordered list of inscribes, sales and transfers of all sats.

query Parameters
sort_by
required
string
Value: "ts"
Example: sort_by=ts

Field to sort by

order
required
string
Enum: "asc" "desc"
Example: order=desc

Sort order

offset
required
integer [ 0 .. 5000 ]
Example: offset=0

Offset for pagination

count
required
integer [ 20 .. 100 ]
Example: count=100

Number of results to return

activity_filter
required
integer
Example: activity_filter=7

Filter for activity type

Responses

Production server

https://api.bestinslot.xyz/v3/sats/activity

https://testnet.api.bestinslot.xyz/v3/sats/activity
