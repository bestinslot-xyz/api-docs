Mempool

Get All UTXOs of Bitcoin Wallet

Returns all UTXOs of a given wallet with inscription and runes information. Note that unconfirmed UTXOs block_height is returned as null.

query Parameters
wallet_addr
required
string
Example: wallet_addr=bc1qvhl8k0xu0stk956tjqx6q5ujp6tdyh24v7eumn

Bitcoin wallet address
