Get collections

Returns ordered list of collections with detailed information about them.

Authorizations:
ApiKeyAuth
query Parameters
sort_by
required
string
Example: sort_by=median_number

Field to sort by. Basic tier fields: supply, min_number, median_number, max_number Pro tier fields: listed_count, floor_price, floor_price_ordswap, floor_price_magiceden, floor_price_ordinalswallet, floor_price_gammaio, floor_price_ordynals, floor_price_ordinalsmarket, floor_price_unisat, floor_price_okx, vol_24h_in_btc, vol_7d_in_btc, sale_cnt_7d, vol_total_in_btc, sale_cnt_total, marketcap

order
required
string
Enum: "asc" "desc"

Sort order

offset
required
integer >= 0

Pagination offset

count
required
integer [ 20 .. 100 ]

Number of records to return

Responses

Get Collection Activity

Returns ordered list of inscribes, sales and transfers of a specific collection.

Authorizations:
ApiKeyAuth
query Parameters
sort_by
required
string
Value: "ts"

Field to sort by

order
required
string
Enum: "asc" "desc"

Sort order

offset
required
integer [ 0 .. 5000 ]

Pagination offset

count
required
integer [ 20 .. 100 ]

Number of records to return

slug
required
string
Example: slug=degods

Collection slug

activity_filter
required
integer
Example: activity_filter=7

Filter for activity types (1=inscribed, 2=transferred, 4=sold, combine with OR operator)

last_new_satpoint
string
Example: last_new_satpoint=32b63c4844db4f704a8df0cf27fcf6ee966a716ea5410e598fe40734b1a7027f:1:0

Use this with offset 0 to continue from a known transfer
