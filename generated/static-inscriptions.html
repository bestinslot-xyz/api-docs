Get Inscription Activity

Returns ordered list of inscribes, sales and transfers of a single inscription.

Authorizations:
ApiKeyAuth
query Parameters
sort_by
required
string
Value: "ts"

Field to sort by

order
required
string
Enum: "asc" "desc"

Sort order

offset
required
integer [ 0 .. 5000 ]

Pagination offset

count
required
integer [ 20 .. 100 ]

Number of records to return

inscription_id
required
string
Example: inscription_id=2f1d51c67998d981055e94dba30d3d1199aa9714049801d7cd35c72066c586abi0

Inscription ID

activity_filter
required
integer
Example: activity_filter=7

Filter for activity types (1=inscribed, 2=transferred, 4=sold, combine with OR operator)

Responses
